## 位置预测评估

本质就是 loc_pred 和 loc 间的对比。

```
{
	len_pred: 该模型得出的预测位置的不同情况的数量，即loc_pred的大小
	uid1: {
		trace_id1: {
			loc_true: 一个数值，代表真实位置的`编号`
			loc_pred: 一个列表集合/有序序列，列表中的元素是 [模型预测出的位置`编号`]。按照预测可能性的大小排列，最有可能的位置排在第1位
		},
		trace_id2: {
			...
		},
		...
	},
	uid2: {
		...
	},
	...
}
说明：关于位置的表示，参考DeepMove模型的输入格式，会对所有的位置进行独热编码，所以每个位置会有一个编号；
若是其他模型没有对位置的编号，直接把位置的具体坐标传进来应该也可以，因为评估本质上就是预测和实际结果的一个对比然后算一个分数。
```
